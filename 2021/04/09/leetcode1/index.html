1、螺旋矩阵 II

题解原文链接（java击败100%）

题目：

给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

示例 1：

输入：n = 3
输出：[[1,2,3],[8,9,4],[7,6,5]]

思路就是每次添加当前的数，同时用一个数组isVisited记录已经到达过的点的位置，然后判断这个方向（初始方向向右）还有没有新点

如果没有的话，按照顺时针调整方向（point和pair[]），循环往复，直到计数到n^2结束

其中方向point的切换要注意
